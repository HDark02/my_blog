# Professional Resume Website

## Description

This is a personal resume website showcasing my professional experience, skills, and projects. The website is built using Flask and features a clean, responsive design that highlights my background as a software engineer.

## Features

- **About Me**: A section that provides a brief introduction and background.
- **Experience**: A detailed list of professional experiences and roles.
- **Skills**: A categorized list of technical and soft skills.
- **Projects**: A showcase of notable projects with descriptions and technologies used.
- **Contact**: A section for visitors to reach out via email or social media.

## Technologies Used

- Flask
- HTML5
- CSS3
- JavaScript
- React
- Tailwind CSS
- Font Awesome

## Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/professional-resume-website.git
   cd professional-resume-website
   ```

2. **Create a virtual environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install the required packages**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**:
   ```bash
   python app.py
   ```

5. **Open your browser** and go to `http://127.0.0.1:5000`.

## Usage

Once the application is running, you can navigate through the different sections of the resume website. Each section is designed to provide a comprehensive overview of my professional background.

## Deployment

For deploying this application, you can use platforms like [Heroku](https://www.heroku.com/) or [PythonAnywhere](https://www.pythonanywhere.com/). Follow their respective documentation for deployment instructions.
